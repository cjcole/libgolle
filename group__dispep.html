Detailed Description

The DISPEP protocol described in Jakobsson M. and Juels A., Millimix: Mixing in Small Batches, DIMACS Technical Report 99-33, June 1999.

The DISPEP protocol leverages the security of Disjunctive Schnorr Identification in order to prove that an El Gamal ciphertext is a re-encryption of one of two difference ciphertexts, without revealing which one.

To use DISPEP, use golle_dispep_setup() to set up the two golle_schnorr_t structures, and then use the results to perform Disjuntive Schnorr proof.
